# Growth Mode Slot

This slot holds configurable growth experimentation strategies for the `growth-loop` agent.

Each config selects:

- a prompt source
- a traffic assumption
- an experiment cadence
- an output shape
- a funnel scope
- a channel focus
- operating roles
- preferred source types
- scoring axes for prioritization

Current configs:

- `codex.json`: live-traffic growth experimentation with `experimenter`, `analyst`, and `conversion` operating roles
